# Abruptly Goblins! Planner

This project is a Python-based automation tool designed to coordinate tabletop gaming sessions for a local comic book store, **The Sorcery Society**. 

I completed this project as part of the **Codecademy Python 3 Curriculum**.

## 📋 How It Works
The script solves scheduling headaches by:
1. **Gathering Data:** Storing gamer names and availability.
2. **Analyzing Attendance:** Finding the night most people can attend using frequency tables.
3. **Optimizing Schedules:** Picking a primary night and a secondary "overflow" night.
4. **Email Automation:** Generating personalized invitation messages.

## 🛠️ Technical Concepts Used
* **Dictionaries & Lists:** For structured data management.
* **Frequency Tables:** To identify peak availability.
* **List Comprehensions:** For efficient data filtering.
* **String Interpolation:** Using `.format()` for custom communications.
